Techs. I've Learned

I have built a solid foundation in core web technologies. These are the tools and languages I use to create responsive, user-friendly websites.

HTML5

Building with semantic elements made structuring web pages clearer and more meaningful. I experienced firsthand, how it improves accessibility and how content is interpreted by browsers

CSS

Styling brought the visual side of web pages to life through colors, spacing, and layout control. It also introduced responsive design for different screen sizes.

Flexbox

Working with Flexbox made alignment and spacing in one direction much easier to handle. It was difficult to grasp at first, but with practice I understood how it simplifies creating flexible and adaptive layouts.

Grid

Grid opened up a more structured way to design two dimensional layouts. It allows precise placement of elements across rows and columns.

Git

Using Git made it easier to track changes and manage code versions over time. It also provides a reliable way to revert mistakes and maintain code stability.

GitHub

Working with GitHub made sharing and storing code online more efficient. It also supports collaboration through repositories and version tracking.
